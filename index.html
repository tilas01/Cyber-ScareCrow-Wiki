Compilation Guide

Cyber-ScareCrow is designed to run securely offline. It consists of a Rust client and a C-based Ring-0 Driver.

Windows Client (Rust)

  • Install Rustup and Visual Studio Build Tools 2022 (Desktop C++).
  • Run cargo build --release in the client directory.

Ring-0 Driver

  • Install the Windows Driver Kit (WDK) and Windows SDK.
  • Open the x64 Native Tools Command Prompt.
  • Run msbuild /p:Configuration=Release /p:Platform=x64.
  • Enable Test-Signing: bcdedit /set testsigning on and reboot.
