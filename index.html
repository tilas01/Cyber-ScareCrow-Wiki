🛡️ Core Protection Features

  • 100% Offline Architecture: No telemetry, no licensing, no remote tracking. Runs entirely on-device.
  • Virtual Machine Detection: Utilizes RDTSC timing spoofing to detect if it's running inside a hostile hypervisor or malware sandbox.
  • Anti-RAT Video Redirection: Ring-0 driver intercepts illicit screen capturing and VNC software, looping a black screen or custom decoy image back to the attacker.
  • Self-Healing: Actively monitors memory for debuggers and reverse-engineering tools scanning for its signature.

⚙️ Compilation & Setup Guide

Cyber-ScareCrow is designed to be reproducibly built from source. You will compile the Rust User-Mode Client and the C-Kernel Driver.

1. Windows Client (Rust)

The client orchestrates the telemetry parsing and coordinates with the driver.

# Install Rustup & VS Build Tools 2022
cd src\client
cargo build --release
# Output: target\release\cyber-scarecrow.exe

2. Ring-0 Kernel Driver (C / WDK)

The kernel driver handles absolute process isolation and un-bypassable deep hooks.

# Install Windows Driver Kit (WDK)
# Open x64 Native Tools Command Prompt
cd src\driver
msbuild /p:Configuration=Release /p:Platform=x64

# Enable Test-Signing Mode
bcdedit /set testsigning on
# Reboot required.

# Load the Driver
sc create ScareCrowDriver type= kernel binPath= "C:\path\to\driver.sys"
sc start ScareCrowDriver
