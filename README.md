## About This Repository
This repository serves a dual purpose:
1. It hosts the **Official Website** (`index.html`) using a customized Gruvbox Dark aesthetic via GitHub Pages.
2. It stores all MarkDown documentation files for the Cyber-ScareCrow ecosystem, including compilation guides and feature breakdowns.

## Documentation Index
The following guides are available in this repository:

*   **[Development & Compilation Guide](DEVELOPMENT_GUIDE.md):** Detailed instructions on how to set up Rust, the WDK, and compile both the user-mode client and kernel-mode driver from scratch.
*   **[Core Features Breakdown](FEATURES.md):** A comprehensive look at the mathematical and technical implementations behind the Anti-RAT video redirection, RDTSC hypervisor polling, and Ring-0 evasion.
